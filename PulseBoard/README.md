# PulseBoard 🚀

PulseBoard is a premium, high-performance, and fully responsive SaaS admin dashboard designed with pure HTML5 and Custom CSS3. Built with modern dark UI aesthetics, it showcases advanced layout systems (CSS Grid, Flexbox, Nested Grids) and interactive behaviors—**without using a single line of JavaScript or external libraries**.

## Project Overview

PulseBoard is a responsive analytics portal concept designed to practice production-ready CSS architectures. It provides a visual showcase of a SaaS analytics panel, complete with custom CSS charts, event schedulers, timelines, custom scrollbars, and fluid layout adjustments.

---

## Features

1. **CSS Checkbox Hack Navigation:** Uses a zero-JS checkbox toggle for sidebar slide-ins on mobile and tablet resolutions.
2. **Interactive CSS-Only Analytics Chart:** Monthly revenue vs expenses chart using CSS Grid columns, vertical height bounds, and hover-triggered CSS tooltips.
3. **Storage Allocation circular progress widget:** A pure CSS conic-gradient visual representation showing 75% cloud allocation.
4. **Zebra-Striped Orders Log:** Beautiful tabular details showing customer logs, amounts, date logs, and glowing custom status badges.
5. **Static CSS Grid Calendar:** A custom month grid highlighting active scheduling, weekend styling, and hover-active event tooltips.
6. **Animated CSS Progress Bars:** Revenue breakdowns utilizing standard transition keyframes on initial render.
7. **Team Collaborators & Timeline Feeds:** Clean Flexbox structures detailing member statuses (online, offline, busy) and a chronological timeline tracker.

---

## Technologies Used

- **HTML5:** Semantic architecture (`<aside>`, `<main>`, `<header>`, `<footer>`, `<section>`).
- **CSS3:** Custom Variables, CSS Grid, Nested Grids, Flexbox, Keyframe Animations, Pseudo-elements, Transitions, and Custom Scrollbars.
- **Remix Icon v4.2.0:** Vector iconography loaded via CDN.
- **Poppins Google Font:** Modern, clean typography.

## Responsive Design

PulseBoard is optimized for the following breakpoints:
- **Desktop (1200px+):** Full multi-column dashboard, sticky left sidebar, optimal whitespace spacing.
- **Laptop (992px–1199px):** Adaptive grids wrapping content cleanly to fit narrower viewports.
- **Tablet (768px–991px):** Collapsed sidebar toggled via top-left hamburger menu. Grids adapt to two columns.
- **Mobile (below 768px):** Spacing compresses, avatars adjust, breadcrumbs hide, and panels stack in a single-column layout.

## Learning Outcomes

- Mastering layout orchestration using **CSS Grid** alongside nested elements.
- Implementing interactive UI patterns (mobile drawers, hover tooltips) using **CSS-only hacks**.
- Managing custom color configurations via **CSS Variables**.
- Designing responsive, scrollable content containers to prevent vertical/horizontal overflow.

## Installation Steps

1. Clone or download the folder to your computer.
2. Ensure you have the `dashboard_avatar.png` asset in the root directory.
3. Open `index.html` directly in any web browser to view the application.
