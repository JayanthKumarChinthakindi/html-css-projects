CSS Timeline

Tracing key milestones in the evolution of CSS visual modeling and native web animations.

2009

CSS Transitions

Introduced the transition properties, enabling developers to interpolate property shifts on states like hover smoothly without JS scripts.

2011

CSS Animations

Standardized the @keyframes spec, offering complex, multi-stage, continuous cycles of animations directly on the web browser thread.

2012

3D Transforms

Added perspective properties and axis rotations, allowing web layouts to break flat planar geometries into full 3D environments.

2015

CSS Custom Properties

Released CSS variables support, which allowed developers to dynamically update, inherit, and animate customized variables.
