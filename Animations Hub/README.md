# Animations Hub

Animations Hub is a premium, modern, fully responsive dark-themed animation gallery built entirely with semantic HTML5 and custom CSS3. It showcases advanced UI animation patterns, keyframe effects, transitions, and layout flows—all without a single line of JavaScript.

## Preview

- **Live Demo**: [animations-hub-jayanth.netlify.app](https://animations-hub-jayanth.netlify.app)
- **GitHub Repository**: [html-css-projects](https://github.com/JayanthKumarChinthakindi/html-css-projects)

## Features

- **Pure HTML5 & CSS3**: Powered strictly by standard HTML elements and custom styling. Zero JS or external animation scripts.
- **No JavaScript**: Navigation dropdown triggers, interactive hover borders, page transitions, and card flipping logic are all driven by CSS selectors, checkboxes, and pseudo-elements.
- **Fully Responsive Design**: Fluid grids and custom responsive breakpoints scale layouts beautifully across mobile, tablet, laptop, and large desktop screens.
- **Modern Dark Theme**: A premium SaaS/Developer tools aesthetic using a rich, dark slate-blue palette and glow parameters.
- **CSS Grid & Flexbox Layouts**: Organized galleries using modern layout algorithms for clean alignments.
- **CSS Keyframe Animations**: Custom `@keyframes` rules driving spinners, pulses, equalizers, orbiting stars, and floating cards at 60 FPS.
- **Hover Effects**: Interactions including image zoom, drawing card outlines, text-shadow pulses, linear shines, and glass distortions.
- **Morphing Buttons**: Distinct button interactions including sliding overlay fills, scale ripples, diagonal gradient shifts, and outline traces.
- **Floating Cards**: Background info panels displaying subtle continuous float cycles.
- **Glassmorphism UI**: Beautiful panels featuring blur parameters (`backdrop-filter`) and low-opacity borders.
- **Animated Progress Bars**: Linear skill metrics that expand dynamically on page load.
- **Responsive Navigation**: Glassmorphic sticky bar that collapses into a stacked drawer on small screen sizes.
- **Clean & Maintainable Code**: Commented sections, meaningful class names, and reusable variables.

## Built With

- HTML5 (Semantic Structure)
- CSS3 (Visual Layout & Staggered Animations)
- CSS Grid (Responsive Multi-Column Alignment)
- Flexbox (Centering & Distribution Alignment)
- CSS Animations (Keyframe Timelines)
- CSS Transforms (Axis Rotations, Scaling, Translations)
- CSS Transitions (Smooth State Shifts)
- Pseudo-elements (`::before` / `::after` fills & borders)

## Getting Started

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/JayanthKumarChinthakindi/html-css-projects.git
   ```
2. Navigate into the animations folder:
   ```bash
   cd "Animations Hub"
   ```
3. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
