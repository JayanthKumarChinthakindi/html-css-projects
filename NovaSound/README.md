# Responsive Product Landing Page

[![Netlify Status](https://api.netlify.com/api/v1/badges/0992fafd-fe67-4961-93f9-2dc45301e4d6/deploy-status)](https://novasound-jayanth.netlify.app)

NovaSound is a sleek, modern, and fully responsive product landing page built to showcase audio equipment. Designed with a clean aesthetic, it serves as a showcase for practicing HTML5 semantic structure, advanced CSS3 styling, and modern responsive layouts.

🚀 **Live Demo:** [https://novasound-jayanth.netlify.app](https://novasound-jayanth.netlify.app)

---

## ✨ Features

- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop viewports.
- **Sticky Navigation Bar**: Keeps main navigation accessible as users scroll the page.
- **Modern Hero Section**: An eye-catching introduction featuring bold typography and key call-to-actions.
- **Feature Cards**: Clean grids highlighting the main selling points of the product.
- **Product Showcase**: Detailed visuals demonstrating the premium build and design of the product.
- **Benefits Section**: Clear breakdown of what makes NovaSound the ideal choice for music lovers.
- **Customer Testimonials**: Social proof styling with user avatars and feedback cards.
- **Pricing Cards**: Visually distinct pricing options with clear calls to action.
- **FAQ Section**: Clean accordion-style list for common user queries.
- **Call-to-Action Section**: Prominent final section designed to drive conversions.
- **Clean Footer**: Organized footer layout with navigation links, social icons, and copyright details.
- **Semantic HTML5**: Written using accessible, structured elements.
- **Pure CSS3**: Styling achieved entirely without bulky external frameworks.
- **Flexbox Layouts**: Used for navigation, list elements, and flexible card distributions.
- **CSS Grid**: Implemented for two-dimensional card layouts and responsive alignments.
- **Mobile-Friendly**: Optimized touch-targets and layout scale adjustments for all devices.

---

## 🛠️ Built With

- **HTML5** - For clean, semantic structure and page elements.
- **CSS3** - For modern layouts, styling, typography, and hover effects.
- **Flexbox** - For flexible, one-dimensional item alignment.
- **CSS Grid** - For structured, two-dimensional grid layouts.
- **Media Queries** - For seamless breakpoint adaptations across all screen sizes.
